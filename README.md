## About

This portfolio is designed to help patients find and connect with Dr. Praveen Thakur online. It highlights his expertise in neurosurgery and spine treatments, making it easy for patients to book appointments and get directions to City Hospital, Kangra.

## Sections

- **Hero** — Introduction with specialization and call-to-action
- **About** — Doctor's background and philosophy
- **Services** — Treatments and specializations offered
- **Experience** — Professional journey and qualifications
- **Testimonials** — Patient reviews and feedback
- **Contact** — Location, phone, and appointment booking
- **Map** — Google Maps direction link to City Hospital

## Deployment

This project is deployed via **GitHub Pages**.

To deploy your own copy:

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://your-username.github.io/repo-name`

Just open `index.html` in any browser — no setup required.
